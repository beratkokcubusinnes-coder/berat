# PromptFind

A premium AI prompt discovery platform built with Next.js and Tailwind CSS.

## Features implemented
- **Dark Mode UI**: sophisticated deep blue/black theme.
- **Interactive Prompt Cards**: Before/After image comparison sliders.
- **Responsive Layout**: Sidebar for desktop, optimized for varied screen sizes.
- **Mock Data**: Pre-populated with trending prompts for UI verification.

## Tech Stack
- **Framework**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS v4
- **Icons**: Lucide React
- **Animation**: Framer Motion

## Getting Started

1. Install dependencies (if not done):
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```
3. Open [http://localhost:3000](http://localhost:3000)

## Project Structure
- `components/layout`: Sidebar, Header
- `components/ui`: Reusable UI components (PromptCard, FilterList)
- `lib/data.ts`: Mock data for development
